# Ejercicio 1

Para cada uno de los siguientes pares de tipos `S` y `T` indicar si están en relación de subtipado, es decir, si `S` ≤ `T` o `T` ≤ `S`, o no están relacionados.

## a)

```
S = !int.end
T = !float.end
```

```
               --------- [S-End]
int ≤ float    end ≤ end
------------------------ [S-OutS]
!float.end ≤ !int.end
```

```
T ≤ S
```

```

```
